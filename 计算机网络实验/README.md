## 计算机网络专题实验

**授课教师**：张利平、郑卫斌

**课程内容**：一共有七次实验，涵盖以下七个主题：实验环境及ARP协议分析、DNS与HTTP协议分析、路由器和交换机、路由协议分析、网络安全与防火墙、传输层TCP协议分析、网络编程。每次上课时间四个小时，需要签到。其中，实验3-5是小班授课，五人一组，无法代签/跑路。

**成绩组成**：70%报告（实验1-7的占比分别为：11%，11%，7%，7%，7%，11%，16%）+30%验收考勤（只有小班授课和最后一次实验需要验收）。

**课程评价**：实验1-6基本上跟着指导书做就能做完，没啥难度。实验7需要验收且占比高，是从10个题目里选一个来做，需要对做的项目有一定的了解，不然会被问住（张利平老师人挺好的，我Vibe出来的的程序完全不了解也给我验收通过了）。总体来说课程质量尚可，能略微加深一些对计网的了解。另：想拿高分需要完成选做。
