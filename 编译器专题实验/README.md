## 编译器专题实验

**授课教师**：戴惠珺

**课程内容**：该课程是形式语言与编译课程的配套实验。课程一共有八次实验，实验任务就是完成一个自己的编译器。实验基本路线是词法分析-语法分析-语义分析-中间代码生成-生成可执行代码，其中前六次实验需要完成实验报告。验收安排在第六次实验之后，去验收时，你的编译器需要完成到实验六结束的程度（即可以生成中间代码）。验收总体很轻松，不涉及代码层面的问答。注意：想验收拿高分务必完成选做与可视化，报告同理。

**成绩组成**：70%报告（六次报告取平均分）+20%验收+10%考勤

**课程评价**：有AI工具辅助的情况下，该实验难度不算高，其带来的结果就是确实也学不到啥东西（如果打算手搓编译器那难度和收获绝对是拉满）。值得注意的一点是：如果借助AI完成这门课的实验，不论是完成编译器还是写报告，可以让你对编译的过程有基本的理解，这会对形式语言与编译这门课的学习有一些帮助。戴老师人很不错，所以不会在验收上为难你。
