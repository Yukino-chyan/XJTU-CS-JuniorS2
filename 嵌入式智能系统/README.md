## 嵌入式智能系统

**授课教师**：李保红

**课程内容**：课程主要讲述ARM指令系统以及基于S3C2440芯片的嵌入式系统相关知识。一共需要完成两次作业（有一定难度，需要一定时间完成）和一次实验（12人一组，总体难度不大）。如果需要了解具体作业内容，可以在仓库里找到。另：若ADS无法下载可以参考我的作业报告以解决。

**成绩组成**：40%平时成绩（考勤、作业）+60%期末成绩

**课程评价及经验**：

课程总体质量不错，李保红老师讲课逻辑是很清晰的。如果有学习过汇编语言与计算机组成原理，学起来会更加轻松。老师在考前会发一个复习PPT（仓库里有），考试只会涉及复习PPT里的内容。

需要注意的是：复习PPT里的内容几乎或多或少都会在考试里有所涉及，比如SDRAM相关的知识、UART、WATCHDOG这些东西可能在复习的时候不会重视，但是都会考到！好消息是，不会考使用第六章学习的内容来编程，编程题考察的都是较为基础的知识点。

作为与汇编强相关的课程，想要学好确实少不了背诵的部分，但是李保红老师出的题能看出来的确想考察理解的重要性。期末的试卷全是大题，我印象比较深刻的是最后一道大题，给了一个很长的程序，涉及中断散转、GPIO、UART相关的寄存器，对程序中的一些语句作用进行考察。所以想要获得高分，背诵和理解两者缺一不可。总体来说老师给分还是很不错的。
