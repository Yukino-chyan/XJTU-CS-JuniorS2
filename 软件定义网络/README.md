## 软件定义网络

**授课教师**：张鹏

**课程内容**：课程讲了软件定义网络相关的知识，需要在虚拟机或Linux操作系统上完成四次Lab。四次Lab涉及的内容分别是：Mininet基础操作、BGP+ECMP相关知识的运用、OpenFlow、VeriFlow。若想了解具体的Lab内容，可以阅读仓库中每个Lab的指导书。

**成绩组成**：10%平时成绩、90%作业+Lab（作业10%，四次Lab分别为10%、20%、20%、30%）。 每个Lab的成绩由报告成绩（0-20）×验收成绩组成（0-6，能给到6分的很少，一般都会给3分以上），满分为100，最后加权到总成绩。

**课程评价及经验**：

这门课上课我没怎么认真听，但还是顺利完成了各个Lab，所以我的评价就是比较考验你的自学能力。每次Lab的验收非常硬核，大概会问20多分钟关于你的工作内容，不仅涉及理论还涉及源码，所以即使该课程不考试，你还是要在每次验收之前完全掌握Lab相关的知识的。

10分的平时分老师都会给满（即使有缺勤），对于每个Lab，都会有选做部分提供给你，建议如果想拿高分的话一定要完成。如果你的原始分数排名比较靠前老师会给你的最终成绩提高几分。

总体来说ANTS实验室的老师课程质量和人品听说都挺不错，西交的ICS也是他们开的，好评也是不少。
