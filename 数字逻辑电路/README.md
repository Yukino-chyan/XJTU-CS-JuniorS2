## 数字逻辑电路

**授课教师**：王今雨

**课程内容**：分为四个章节，第一章重点学习数制编码、逻辑代数，第二章学习组合逻辑电路的分析与设计以及一些组合逻辑器件，第三章学习时序逻辑电路的分析与设计以及相应的器件，第四章学习可编程逻辑器件。

**成绩组成**：30%平时成绩（会有随堂测试）、70%期末。

**课程评价及经验**：

这门课我是重修的，大二试图一天从0开始速通完美大失败，希望大家不要重蹈我的覆辙。另：对于重修的学生没有平时分，还是略有些难以通过的。

下面说说考试以及四章的重点是什么：

首先关于考试形式：填空题15选12，选择题7选5，简答题7选5，分析题2选1（每一道里面有两个小问，组合逻辑+时序逻辑）+一道异步脉冲时序分析必选，设计题一道组合逻辑必选（仅用一片38译码器实现一个4位的最大项表达式），一道同步时序逻辑必选，编程题Verilog二选一（忘了是啥，不太难）。

总体考试难度偏大，且不仅有背诵（印象中很怪的一道简答题，建立原始状态表的关键是确定以下三个问题，要你写出来），还有要求你运用的题。想要通过起码要有4天及以上的复习。

**第一章**：数制转换自不必说，BCD码，余三码，2421码都要熟练记住，还有三种校验方式也要熟练运用。基本除了奇偶校验都考了。逻辑代数这一章一定要认真学，而且重在理解！重在理解！还有最后的禁止逻辑和替代尾因子也要会，考试简答题考了禁止逻辑法。1.4则随便看看就好。

**第二章**：分析和设计不用说都要会，跟着PPT学就好。竞争与险象也是一个重点，填空简答选择都考到了相关的内容。你需要掌握险象产生的原因、分类，险象的判别，险象的解除等等知识。关于组合逻辑器件，四种组合逻辑器件都考了，重点要求你熟练运用的是编码/译码器和数据分配/多路选择器。三态门在填空好像有涉及，加法器考了一道简答，要求你对比串行加法器和超前进位加法器。

**第三章**：还是分析和设计都要会。其中异步时序应该是不会考设计的，但是一定会有一道分析。这里第一遍学的时候很难，但是学懂了就能慢慢做题了。关于时序逻辑器件，计数器考了填空和分析题，让写出串行计数器的CLK以及一个74LS163的电路实现的功能；寄存器不太记得了；555也没考，但是建议记一下频率和占空比怎么算就行。值得一提的是，有一道简答题考了D、T、JK触发器的相互构建，值得练习一下。

**第四章**：很恶心的一章，你需要背诵很多东西。考试涉及到第四章的内容比想象中的要多，请记住：不是光记住分类和作用是啥就行了，这只是基础。比如简答题考了FPGA的特性和GAL对比PAL这两道题，填空题有印象的有一次可编程器件有哪些等题目。关于Verilog编程题，选择组合逻辑的题目就行，不算难，不建议丢掉这一部分的分数。
